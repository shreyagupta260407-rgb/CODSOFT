# Simple To-Do List (Python)

This is a basic command-line **To-Do List application** built in Python.  
It helps you add, view, complete, and delete tasks, while saving them in a JSON file for persistence.

---

## Features
- Add new tasks with description and timestamp
- View all tasks in a clean list
- Mark tasks as completed
- Delete tasks
- Save tasks automatically in `my_tasks.json`

---

## Requirements
- Python 3.x

---

1. Download or clone this repository.
2. Open a terminal in the project folder.
