# Contact Book (Python)

This is a simple **Contact Book program** made in Python.  
It lets you save, view, search, update, and delete contacts using a menu system.

---

## Features
- Add a new contact (name, phone, email, address)
- View all saved contacts
- Search for a contact by name
- Update contact details
- Delete a contact
- Exit the program safely

---

## Requirements
- Python 3.x

---

## How to Run
1. Download or copy the code into a file named `contact.py`.
2. Open a terminal or command prompt in the folder.
3. Run the program:
   ```bash
   python contact.py
